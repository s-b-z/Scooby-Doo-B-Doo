👋 About Me  
Scooby-Doo B-Doo is an interactive web experience that blends themed visuals, animation, and audio to create an engaging browser-based project.  

It was designed to explore how real-world browser limitations affect media playback and how user interaction can be used to work within those constraints.

---

✨ Features  

- 🔊 Audio triggered through user interaction  
- 🎬 Themed visuals and animations  
- ⚡ Event-driven JavaScript behavior  
- 🎯 Handling browser media restrictions  

---

🛠️ Tech Stack  

- HTML  
- CSS  
- JavaScript  

---

💡 Purpose  
This project focuses on understanding how browsers handle media and interaction logic. It highlights:

- Event-driven programming  
- Managing audio playback restrictions  
- Creating immersive, interactive experiences  
- Combining visual design with functional behavior
